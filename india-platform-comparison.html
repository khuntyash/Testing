India Platform Comparison for Cropper Hub

Baseline assumption: 1 API service + 4 always-on workers + managed Redis + managed Postgres + object storage + logs. Costs are practical ranges, not billing quotes.

Quick takeaway: Azure is a good option for India. For queue-heavy PDF fan-out, AWS and Azure are safest for reliability; OCI is strongest on cost; Hetzner is cheapest but needs highest ops effort.
